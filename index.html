What is AQUAVIEW?

AQUAVIEW is a new framework that allows for a combination of centralized and distributed data storage and processing.
It's part public repository, part data processing pipeline, and part data management system.
It structures data based on how it's being used by real users...
...importing frequently used data for easy access and compute, while offloading older data to the distributed storage system.
AQUAVIEW also has a robust API infrastructure, which provides users ability to access metadata, data locations, and when appropriate, the data itself.

AQUAVIEW will automagically extract the metadata from your data...

...and combine it with the metadata from other data sources to create a single, searchable, and reusable metadata record.

And can automagically package this data according to one of several standards, including the archive's.
